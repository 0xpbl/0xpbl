# Uma triste história... em desenvolvimento

Este repositório contém documentos que tecnicamente não deveriam existir, mas que existem de qualquer forma por questões de consistência estatística.

**Status atual:** Em superposição entre "quase pronto" e "nunca vai estar pronto".
.
**Última atualização:** A última vez que alguém tentou atualizar isso, o sistema entrou em loop recursivo. Recomendamos não tentar novamente.

**Informações relevantes:** Nenhuma.

**Informações irrelevantes:** O número de arquivos neste repositório é simultaneamente 11 e 0, dependendo do observador. A pasta `thehistory/` pode ou não conter a verdade sobre tudo. O arquivo `qel.png` na pasta `img/` é uma imagem que representa algo, mas ninguém sabe exatamente o quê.

**Aviso legal:** Este repositório não assume responsabilidade por paradoxos temporais, realidades alternativas ou crises existenciais resultantes da leitura dos documentos.
